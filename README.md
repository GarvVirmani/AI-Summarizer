# 📝 AI Summarizer

🚀 **AI Summarizer** is a React-based web application that uses the **Gemini API** to generate concise summaries from user-provided text.

🔗 **Live Demo**: [AI Summarizer](https://garvvirmani.github.io/AI-Summarizer/)

---

## ✨ Features
✅ Summarizes long text in seconds  
✅ Uses **Google Gemini API** for AI-powered summaries  
✅ Simple & user-friendly interface  
✅ Fully responsive design
